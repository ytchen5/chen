## cut工具

cut是列截取工具，用于列的截取

***语法和选项***

***语法：***

```
# cut 选项 文件名
```

***常见选项***

```
-C	以字符为单位进行分割，截取
-d	自定义分隔符，默认为制表符\t
-f	与-d一起使用，指定截取那个区域
```

***举例说明：***

```
cut -d: -f2 /etc/passwd
cut -d: -f1,6,7 /etc/passwd 	#截取1，6，7列
cut -c2 1.txt 					#截取第2个字符
cut -c1-5 1.txt 				#截取第1个到第5个字符
cut -c-5 1.txt					#截取前从开始到第5个字符
cut -c6- 1.txt					#截取第6个字符到最后的字符
```
